switch_cuda.sh

由于饱受cuda环境设置困扰，从网上扒下了这一段代码，写成sh文件放在这里：

function switch_cuda {
   v=$1
   export PATH=/usr/local/cuda-$v/bin:$PATH
   export CUDADIR=/usr/local/cuda-$v
   export CUDA_HOME=/usr/local/cuda-$v
   export LD_LIBRARY_PATH=/usr/local/cuda-$v/lib64:$LD_LIBRARY_PATH
}

使用方法为：

（1）将switch_cuda.sh放在某个目录下，这里以主目录~为例。

（2）使用vim开主目录的.bashrc文件：

sudo apt install vim
sudo vim ~/.bashrc

（3）按i进入编辑模式后，跳到文件末尾，加上这两行（以cuda版本为11.7为例）：

source ~/switch_cuda.sh
switch_cuda 11.7

之后按esc进入命令行模式后输入:wq保存并退出。

（4）关闭当前窗口，当再次打开命令行时，.bashrc会自动生效。若需要在当前窗口生效，可以输入：

source ~/.bashrc
